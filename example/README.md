# Example Videos

This directory contains example videos to demonstrate the AI upscaling capabilities of different models.

## Contents

### `example_video/`

- **`base.mp4`** - Original video (low resolution)
- **`example RealESRGAN_x2plus.mp4`** - Upscaled with RealESRGAN x2 model
- **`example RealESRGAN_x4plus.mp4`** - Upscaled with RealESRGAN x4 model (default)
- **`example RealESRNet_x4plus.mp4`** - Upscaled with RealESRNet x4 model (cleaner)
- **`example RealESRGAN_x4plus_anime_6B.mp4`** - Upscaled with anime-specific model

## Usage

These examples are accessible through the **"üîç Open Video Comparison Modal"** button in the Upscaler tab.

The modal allows you to:
- Compare the original video side-by-side with upscaled versions
- Switch between different models to see the differences
- Synchronized playback of both videos
- Loop functionality for continuous comparison

## Notes

- All example videos are **muted** (no audio)
- Videos maintain their original aspect ratio
- The modal displays videos in fullscreen for detailed quality comparison
